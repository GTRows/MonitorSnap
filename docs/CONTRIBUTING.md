# Contributing

Thanks for considering contributing to Display Presets!

## Getting started

1. Fork the repo
2. Clone your fork
3. Install deps: `pip install -r requirements.txt`
4. Make your changes
5. Test: `python main.py`
6. Submit a PR

## Code style

- Keep it simple
- Use clear variable names
- Comment when necessary (not excessively)
- Follow existing patterns

## What to contribute

- Bug fixes
- New features (discuss in an issue first)
- Documentation improvements
- Performance improvements

## Testing

Before submitting:
- Test on your own multi-monitor setup
- Make sure the tray icon works
- Verify presets save/load correctly
- Check autostart functionality

## Questions?

Open an issue or start a discussion.
