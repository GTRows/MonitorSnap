# Screenshots

This folder contains screenshots for the README.

To contribute screenshots:
1. Take screenshots at 1920x1080 or higher resolution
2. Save as PNG format
3. Name files descriptively:
   - `dashboard.png` - Main dashboard window
   - `tray-menu.png` - System tray context menu
   - `settings.png` - Settings tab
   - `hotkey-assignment.png` - Hotkey configuration
   - `monitor-preview.png` - Monitor layout preview
